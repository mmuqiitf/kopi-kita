# Kopi Kita CRM — Mini CRM + AI Global Promo Helper

Kopi Kita CRM is a full-stack Next.js app for Mimi’s Coffee Shop to manage customers, track interests, generate weekly global promo ideas with AI, and chat using current store data.

## Tech Stack

- Next.js 16 (App Router, TypeScript)
- Drizzle ORM + PostgreSQL (Supabase-compatible)
- NextAuth credentials authentication
- TanStack Query for client data syncing
- Vercel AI SDK + Groq for AI generation/chat
- Tailwind + shadcn/ui (Lyra style, amber theme)

## Features

- Authentication with protected app routes
- Customer CRM:
  - add / edit / delete
  - search by name/contact
  - filter by interest tags
  - multi-select interest tags with create-on-input
- Promo Ideas (AI core):
  - generates 3 campaign ideas from aggregate customer trends
  - includes segment, why-now, message CTA, and optional best time
  - saves generated campaigns to DB
  - lists this week’s campaigns
- Dashboard:
  - total customers
  - top products and top interests
  - this week’s campaign suggestions + copy message action
- AI Chat:
  - answers using live aggregate data context
  - protected API route (authenticated only)

## Local Setup

### 1) Install dependencies

```bash
npm install
```

### 2) Create `.env.local`

```bash
DATABASE_URL=postgres://...
NEXTAUTH_SECRET=replace-with-secure-random-string
NEXTAUTH_URL=http://localhost:3000
GROQ_API_KEY=your_groq_api_key
```

### 3) Run DB migrations + seed

```bash
npm run db:setup
```

### 4) Run app

```bash
npm run dev
```

Open http://localhost:3000
